# welcome to ongoing project

## how to run the Frontend code?
### follow the steps :
#### first look at your folder structure then go inside the folder, open it in VS CODE
```
cd "your-project_name"
```
### Next 

```
npm install
```

## To run the UI
```
npm run dev
```

### Now for Backend part 
### Steps : 

### first get into the directory
```
cd Api
```
### then
```
pip install psycopg2-binary fastapi uvicorn sqlalchemy pydantic pandas
```

### To run the code (API):
```
uvicorn app:app --reload
```
### OR
```
python app.py
```
